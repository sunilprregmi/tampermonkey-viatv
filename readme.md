# ViaTV Reseller WebLogin

A Tampermonkey extension that allows customers of specific ViaTV resellers to access the web login portal by redirecting the `reseller_list` API to a custom GitHub-hosted source.

## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) extension.
2. Click the link below to install the script directly:
**[Install via_tam.user.js](https://github.com/sunilprregmi/tampermonkey-viatv/raw/main/via_tam.user.js)**
3. Confirm by clicking **Install** in the Tampermonkey tab.

---

## Features

* **Compatibility:** Works with both `Fetch` and `XMLHttpRequest` (XHR).
* **Automatic Updates:** Syncs directly with this repository.
